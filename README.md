# ⚡ Renewable Energy Forecasting using Advanced Excel & PostgreSQL

---

## 📌 Project Overview

Renewable energy plays a vital role in sustainable development and reducing dependence on conventional energy sources.

This project analyzes renewable energy generation data along with weather conditions using Advanced Excel and PostgreSQL to understand how environmental factors influence energy production.

The project transforms raw renewable energy data into meaningful insights that can support monitoring, planning, forecasting, and data-driven decision-making.
